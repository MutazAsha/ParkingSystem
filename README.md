# ParkingSystem
# A full parking management system built using React, Redux, Tailwind CSS, and Vite on the frontend, and .NET Core MVC on the backend with full API integration, authentication handling, and dependency injection.
This system allows managing parking slots, users, vehicle entries/exits, and provides a smooth UI/UX with fast performance.

# Skills : React , Redux Toolkit , Tailwind CSS , Vite , .NET Core , MVC Architecture , REST APIs , Dependency Injection , CORS Configuration for React
